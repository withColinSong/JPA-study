# 2. 단방향 양방향
> 테이블
- 외래 키 하나로 양쪽 조인 가능
- 사실 방향이라는 개념이 없다.

> 객체
- 참조용 필드가 있는 쪽으로만 가능
- 한쪽만 참조하면 단방향
- 양쪽이 서로 참조하면 양방향

> 연관관계의 주인
- 연관관계의 주인 : 외래 키를 참조하는 참조
- 주인의 반대편 : 외래 키의 영향을 주지 않고, 단순 조회용
